# نظام إدارة الطلاب (Student Management System) - Flutter

تطبيق احترافي لإدارة بيانات الطلاب مبني باستخدام إطار العمل **Flutter** وقاعدة بيانات **SQLite** المحلية. التطبيق مصمم خصيصاً ليعمل بدون إنترنت مع دعم كامل للغة العربية.

## المميزات الرئيسية
- **إدارة الطلاب:** إضافة، بحث، تعديل، وحذف بيانات الطلاب.
- **قاعدة بيانات محلية:** تخزين آمن للبيانات باستخدام SQLite.
- **تصدير PDF:** إنشاء تقارير منظمة للطلاب تدعم اللغة العربية.
- **النسخ الاحتياطي:** إمكانية تصدير قاعدة البيانات كملف ومشاركتها واستعادتها لاحقاً.
- **واجهة مستخدم حديثة:** تصميم جذاب يدعم اتجاه اليمين إلى اليسار (RTL).

## المتطلبات التقنية
- Flutter SDK: >= 3.0.0
- Dart: >= 3.0.0
- قاعدة البيانات: sqflite

## طريقة التشغيل وبناء التطبيق
1. بعد تحميل المشروع، قم بتشغيل الأمر التالي لتحميل المكتبات:
   ```bash
   flutter pub get
   ```
2. لبناء ملف APK للنسخة النهائية:
   ```bash
   flutter build apk --release
   ```
3. ستجد الملف في المسار: `build/app/outputs/flutter-apk/app-release.apk`

## كيفية الرفع على GitHub (من الجوال)
بما أنك تستخدم الجوال، يمكنك رفع ملف `student_management_system.zip` مباشرة إلى مستودعك الجديد على GitHub، أو استخدام تطبيقات مثل **MGit** لرفع الملفات بشكل منفصل.
